Tic Tac Duck Hunt

Instructions

  • Shoot ducks to mark a square
  • Hitting the dog is an instant loss
  • Mark a square before the timer runs out
  • Each duck hit results in 5 - 20 points depending on how quickly it is shot
  • If game results in a tie, highest score wins

Select board size, number of ducks in a row to win, and press START to play!
